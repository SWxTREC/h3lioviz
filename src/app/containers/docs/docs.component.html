Next Generation 3D Solar Wind Interactive Data Visualizations

This web site is part of an initial deployment and proof of concept to:

  • develop an initial application to enable full 3D visualization of Dusan Odstrcil's Enlil solar wind model
  • deploy and run the Enlil model in the SWx TREC Cloud-based computing environment alongside the visualization tool
  • bring about effective collaborative use in forecasting offices

Further features and enhancements to come. Learn more and get in touch via the SWx TREC site.

Keyboad tips for panning:

shift + left-click
only allows rotation about the center
shift + right-click
pan/move the image around by its center
ctrl + right up/down
zooms in/out
