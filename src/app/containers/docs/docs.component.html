Next Generation 3D Solar Wind Interactive Data Visualizations

This website is part of an initial deployment and proof of concept to:

  • develop an initial application to enable full 3D visualization of solar wind model output
  • deploy and run the solar wind models in the SWx TREC Cloud-based computing environment alongside the visualization tool
  • bring about effective collaborative use in forecasting offices

Further features and enhancements to come. Learn more and get in touch via the SWx TREC site.
